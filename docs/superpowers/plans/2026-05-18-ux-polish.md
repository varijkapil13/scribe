## Task 4 — Wire sidebar search to filter notes

**Files:**
- Modify: `Scribe/UI/MainWindow/MainWindowView.swift`

Context: `.searchable(text: $searchText, ...)` creates a native macOS search field in the sidebar, but `searchText` is never consumed. When the user types, nothing happens. The fix: when `searchText` is non-empty, replace all sidebar sections with a filtered list of matching notes (title + body search). When empty, show the normal sidebar.

The `allNotes` array is already maintained at `@State private var allNotes: [Note]` and kept up-to-date via `NoteStore.shared.observeAllNotes()`.

- [ ] **Step 1: Add `sidebarSearchResults` computed property**

After the `scheduleTagReload()` function (around line 388), add:

```swift
private var sidebarSearchResults: [Note] {
    guard !searchText.isEmpty else { return [] }
    let query = searchText.lowercased()
    return allNotes
        .filter {
            $0.title.lowercased().contains(query) ||
            $0.body.lowercased().contains(query)
        }
        .sorted { $0.updatedAt > $1.updatedAt }
}
```

- [ ] **Step 2: Wrap sidebar content in a search-active guard**

In `MainWindowView.swift`, the `sidebar` computed var starts at line 187 with `List(selection: $selection) {`. The current content is a series of `Section` blocks.

## Task 8 — Add focus affordance to note title field

**Files:**
- Modify: `Scribe/UI/Notes/NoteDetailView.swift`

Context: The title `TextField` uses `.plain` style with no visual cue that it's editable — clicking it works, but there's no pre-focus affordance. Adding a `@FocusState` + a subtle `.accentColor.opacity(0.07)` rounded background when focused matches how native macOS apps like Notion and Craft handle editable headings, without breaking the editorial aesthetic.

- [ ] **Step 1: Add `@FocusState` for title field**

In `NoteDetailView`, after the existing `@State` declarations (around line 13), add:

```swift
@FocusState private var titleFocused: Bool
```

- [ ] **Step 2: Apply focus state and background to the title TextField**

Find the title `TextField` (around line 30):

```swift
TextField("Untitled", text: Binding(
    get: { vm.note.title },
    set: { vm.note.title = $0; vm.markDirty() }
))
.font(.system(size: 20, weight: .semibold, design: .serif))
.textFieldStyle(.plain)
.foregroundStyle(.primary)
```

Replace with:

```swift
TextField("Untitled", text: Binding(
    get: { vm.note.title },
    set: { vm.note.title = $0; vm.markDirty() }
))
.font(.system(size: 20, weight: .semibold, design: .serif))
.textFieldStyle(.plain)
.foregroundStyle(.primary)
.focused($titleFocused)
.padding(.vertical, 2)
.padding(.horizontal, 4)
.background(
    RoundedRectangle(cornerRadius: DesignTokens.Radius.xs, style: .continuous)
        .fill(titleFocused ? Color.accentColor.opacity(0.07) : .clear)
        .animation(.easeOut(duration: DesignTokens.Motion.fast), value: titleFocused)
)
```
